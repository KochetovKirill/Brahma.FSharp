Brahma.FSharp is Brahma-based library for FSharp quotations to OpenCL translation.

more information: https://sites.google.com/site/semathsrprojects/home/brahma-fsharp

NuGet package: https://nuget.org/packages/Brahma.FSharp/

G+ community: https://plus.google.com/u/0/communities/111965488041706059072
-------------------------------------------------------

Brahman.OpenCL is Brahma.FSharp-based library which provide GPU-accelereted solutions for end users.

Brahman.Substrings is a part of Brahman.OpenCL. This library provide GPU-accelerated algorithms for substrings matching.

more information: https://sites.google.com/site/semathsrprojects/home/brahman-opencl/brahman-substrings

NuGet package: https://nuget.org/packages/Brahman.Substrings/
